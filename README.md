# 苏东坡人生参照系

> 你不需要立刻振作，也不必假装豁达。

---

公元1080年，苏东坡刚到黄州。

他在定惠院的小庙里昏睡了一整天，晚上才勉强出门散步。后来他写了首《卜算子》，里面有一句："寂寞沙洲冷"。

冷到骨子里的那种冷。不是什么"苦难是财富"，不是什么"一切都会好起来"。就是冷。

他没有逼自己立刻振作。他花了三年，才从"死灰吹不起"走到"一蓑烟雨任平生"。
